---
title: "Webhooks"
description: "Endpoints for managing webhooks and deliveries"
---

## List Webhooks
- GET `/webhooks`

## Get Webhook
- GET `/webhooks/{id}`

## Create Webhook
- POST `/webhooks`

```json
{
  "name": "Success Notification",
  "url": "https://example.com/webhook",
  "eventType": "success",
  "headers": { "Authorization": "Bearer token" },
  "isActive": true
}
```

## Update Webhook
- PATCH `/webhooks/{id}`

## Delete Webhook
- DELETE `/webhooks/{id}`

## List Deliveries
- GET `/webhooks/deliveries?webhookId=wh_123&page=1&limit=50`

<Tip>
Verify webhook signatures using your secret and HMAC-SHA256.
</Tip>
