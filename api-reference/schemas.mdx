---
title: "Schemas"
description: "Endpoints for managing schemas and data validation"
---

## List Schemas
- GET `/schemas`

## Get Schema
- GET `/schemas/{id}`

## Create Schema
- POST `/schemas`

```json
{
  "name": "Employee Schema",
  "description": "Employee data",
  "fields": [
    { "name": "id", "type": "string", "required": true },
    { "name": "email", "type": "string", "required": true }
  ]
}
```

## Update Schema
- PATCH `/schemas/{id}`

## Delete Schema
- DELETE `/schemas/{id}`

## Validate Data
- POST `/schemas/{id}/validate`

```json
{ "id": "123", "email": "john@example.com" }
```

<Tip>
Use `filefeed.schemas.validate({ schemaId, data })` to validate programmatically.
</Tip>
