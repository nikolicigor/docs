---
title: "Going to Production"
description: "Checklist for a reliable FileFeed integration"
---

## Setup
- [ ] Obtain API key (Dashboard → My Account → Security Settings)
- [ ] Create webhook endpoint and register it
- [ ] Store webhook secret securely
- [ ] Implement signature verification

## Configuration
- [ ] Create clients with correct SFTP credentials
- [ ] Define schemas for expected formats
- [ ] Create pipelines with field mappings
- [ ] Upload starter files (headers)
- [ ] Configure webhook URLs

## Production Readiness
- [ ] Test uploads and processing end-to-end
- [ ] Implement retries and idempotency
- [ ] Monitor webhook delivery failures
- [ ] Validate sample data and edge cases

<Tip>
Acknowledge pipeline runs after successful persistence to avoid reprocessing.
</Tip>
