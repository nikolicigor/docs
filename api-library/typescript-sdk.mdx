## Resources and methods

<AccordionGroup>
  <Accordion icon="server" title="pipelineRuns">
    - `list(params)` – filter by status, client, pipeline, paginate
    - `retrieve(id)` – get a single run
    - `getData({ pipelineRunId, offset?, limit? })` – retrieve processed rows
    - `ack({ pipelineRunId })` – mark as processed
    - `reprocess({ pipelineRunId })` – retry processing
    - `getOriginalFileUrl({ pipelineRunId, expiresIn? })`
    - `getProcessedFileUrl({ pipelineRunId, expiresIn? })`

    ```ts
    // Example
    const runs = await filefeed.pipelineRuns.list({ status: 'completed', limit: 25 });
    const data = await filefeed.pipelineRuns.getData({ pipelineRunId: runs.data[0].id, limit: 1000 });
    await filefeed.pipelineRuns.ack({ pipelineRunId: runs.data[0].id });
    ```

    <Tip>
    See API Reference: [/api-reference/introduction](/api-reference/introduction)
    </Tip>
  </Accordion>

## Example: Paginate Data

```ts
const runs = await filefeed.pipelineRuns.list({ status: 'completed', limit: 50 });
for (const run of runs.data) {
  let offset: number | null = 0;
  do {
    const page = await filefeed.pipelineRuns.getData({ pipelineRunId: run.id, limit: 1000, offset });
    // process page.data
    offset = page.data.length === 1000 ? (offset ?? 0) + page.data.length : null;
  } while (offset !== null);
}
```
